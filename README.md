# 🪶 سماء الأدب

موقع لمتابعة حفظ الشباب وقراءتهم لشروح المعلقات الشعرية، مبني بـ React + Vite ومتصل بقاعدة بيانات Supabase.

## المميزات

- **صفحة التسجيل:** يختار الطالب اسمه (مع بحث وإمكانية إضافة اسم جديد)، ثم القصيدة والأسبوع، ثم يعلّم أيام الحفظ وقراءة الشرح (الأحد–الخميس) في شبكة مربعات.
- **صفحة الإحصائيات (عامة):** أربعة رسوم بيانية — الحفظ والشرح لأسبوع محدد، والإجمالي التراكمي للحفظ والشرح منذ بداية القصيدة.
- **لوحة الإدارة (محمية بكلمة مرور `1289`):** عرض المتأخرين عن الحفظ (عمود «يومين فأكثر» وعمود «٤ أيام فأكثر»)، استبعاد/إعادة الطلاب، وإضافة قصائد جديدة.

## خطوات التشغيل

### 1) إنشاء مشروع Supabase
- ادخل إلى supabase.com وأنشئ مشروعاً جديداً (مجاني).
- من **SQL Editor** الصق محتوى الملف `supabase-schema.sql` ثم اضغط **Run**. هذا ينشئ الجداول والبيانات الأولية (معلقة عنترة + أسماء تجريبية).

### 2) تشغيل الموقع محلياً
```bash
npm install
npm run dev
```
عند أول فتح للموقع، ستظهر شاشة إدخال بيانات الاتصال:
- **Project URL** و **anon key** تجدهما في: Supabase ← Project Settings ← API.

(بديل: أنشئ ملف `.env` يحوي `VITE_SUPABASE_URL` و `VITE_SUPABASE_ANON_KEY` لتجاوز شاشة الإعداد.)

### 3) النشر (اختياري)
```bash
npm run build
```
ثم ارفع مجلد `dist/` إلى أي استضافة (Vercel أو Netlify). أضِف متغيري البيئة عند النشر.

## ملاحظات مهمة

- **حساب المتأخرين:** يقارن أيام الحفظ المتاحة (أيام العمل الأحد–الخميس منذ بداية القصيدة حتى اليوم) بالأيام المسجّلة. عند بدء قصيدة جديدة يُعاد الحساب من الصفر تلقائياً.
- **تكرار التسجيل:** يمكن للطالب الدخول يومياً وتعديل أيامه؛ الحفظ يحدّث السجل ولا يكرره.
- **كلمة مرور الإدارة:** `1289` (تُغيّر من المتغير `ADMIN_PASSWORD` في `src/components/AdminPage.jsx`).
- **تاريخ البداية:** الأسبوع الأول يبدأ من 17 يونيو 2026 (بداية معلقة عنترة).
